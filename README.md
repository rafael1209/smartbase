# Smart Base (בסיס חכם) - Frontend Mockup & Local Server

Этот проект представляет собой интерактивный фронтенд-макет портала цифровых услуг **Smart Base (בסיס חכם)**. Макет доработан для бесшовной локальной навигации и поставляется со встроенным легковесным веб-сервером.

---

## 🚀 Особенности / Features

* **Интерактивные переходы / Interactive Navigation:**
  * Клик по карточке подтверждения на главной странице (`index.html`) плавно перенаправляет на страницу детальной информации (`Smart Base (2).html`) с микроанимацией наведения.
  * Клик по кнопке-стрелке «Назад» в верхнем правом углу второй страницы возвращает на главную страницу с интерактивным эффектом при наведении.
* **Встроенный веб-сервер / Built-in Web Server:**
  * Написан на чистом Node.js (без тяжелых внешних зависимостей вроде `express`).
  * Полностью поддерживает UTF-8 кодировку для корректного отображения текстов на иврите/русском.
  * Корректно декодирует URL-адреса с пробелами и спецсимволами.
* **Быстрый запуск / Quick Launch:**
  * Удобный Windows-скрипт (`start-server.bat`) для мгновенного запуска и открытия в браузере в один клик.

---

## 🛠️ Запуск проекта / How to Run

### Способ 1: В один клик (Windows)
Просто дважды кликните по файлу **`start-server.bat`** в папке проекта. 
* Скрипт сам запустит локальный сервер и автоматически откроет страницу в вашем браузере по умолчанию.

### Способ 2: Через терминал (NPM)
1. Откройте терминал в папке проекта.
2. Запустите сервер:
   ```bash
   npm start
   ```
3. Откройте в браузере: **[http://localhost:3000](http://localhost:3000)**

## 📂 Структура проекта / Project Structure

* `index.html` — Главная страница портала со списком заявок и услуг.
* `Smart Base (2).html` — Страница детальной информации по выбранной заявке.
* `server.js` — Легковесный веб-сервер на Node.js.
* `package.json` — Описание проекта и скрипты запуска.
* `start-server.bat` — Скрипт быстрого запуска для Windows.
